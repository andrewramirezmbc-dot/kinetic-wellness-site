# Kinetic Wellness Group — Website

GLP-1 weight loss program website for Kinetic Wellness Group.

## Stack
- Static HTML/CSS/JS (single page)
- No build tools required
- Hosted via GitHub Pages

## Local Development
Open `index.html` in any browser.

## Deployment
Enable GitHub Pages in repo Settings → Pages → Source: `main` branch, root `/`.

Site will be live at: `https://<username>.github.io/kinetic-wellness-site/`

## Images
All images are in `/images/` — sourced from stock photography (Pexels/Unsplash). Replace with real brand photography when available.

## Structure
```
├── index.html          # Main site
├── images/
│   ├── hero-*.jpg      # Hero grid photos (3:4 portrait)
│   ├── pain-*.jpg      # Pain point card photos (16:9)
│   └── step-*.jpg      # Step card photos (4:3)
└── README.md
```
